Ubuntu 18.04 server ens3修改为eth0

2020年5月16日
| linux
在新安装的Ubuntu 18.04 server上执行ifconfig -a命令时，发现熟悉的eth0变成了陌生的ens3。喜旧厌新的我决定将ens3改成熟悉的eth0。 1.修改/etc/default/grub文件 将 GRUB_CMDLINE_LINUX="" 修改为 GRUB_CMDLINE_LINUX="net.ifnames=0 biosdevname=0"。 2.更新grub sudo update-grub 3.重启设备 reboot 重新登录设备，执行ifconfig -a，就会发现ens3变成了eth0。……
