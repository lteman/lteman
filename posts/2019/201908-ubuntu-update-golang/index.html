Ubuntu升级golang 1.12

更新golang版本

  • add-apt-repository ppa:longsleep/golang-backports
  • apt-get update
  • apt-get install golang-go
  • apt install golang-golang-x-tools
  • rm -rf /etc/apt/sources.list.d/longsleep-ubuntu-golang-backports-xenial.list

检查安装golang版本

# go version
go version go1.12.7 linux/amd64
